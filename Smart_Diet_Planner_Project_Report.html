Chapter 1
Introduction
1.1 Project Description

The Smart Diet Planner represents a cutting-edge approach to personal nutrition management, combining artificial intelligence with user-friendly web technology to deliver personalized dietary guidance. In today's fast-paced world, maintaining optimal nutrition has become increasingly challenging due to busy lifestyles, limited knowledge of nutritional requirements, and the complexity of balancing various dietary needs.

This web-based application addresses these challenges by providing an intelligent platform that learns from user preferences, health conditions, and dietary goals to generate customized meal plans and nutritional recommendations. The system incorporates advanced features such as AI-powered meal planning, comprehensive food logging, nutritional analytics, and personalized recommendations based on individual health profiles.

The primary objective of this project is to develop a comprehensive nutrition management system that empowers users to make informed dietary decisions while simplifying the process of meal planning and nutritional tracking. The application serves both regular users seeking to improve their dietary habits and administrators who need to manage the system's food database and user analytics.

Key features of the Smart Diet Planner include user registration and profile management, AI-driven personalized diet plan generation, comprehensive food logging with nutritional tracking, progress monitoring and analytics, admin panel for system management, and integration with Google's Generative AI for intelligent recommendations.

Chapter 2
Literature Survey
2.1 Existing System

Several nutrition management and diet planning applications currently exist in the market, each with varying capabilities and limitations. Popular applications like MyFitnessPal, Lose It!, and Cronometer provide basic calorie tracking and food logging functionalities but lack sophisticated AI-driven personalization.

Traditional diet planning systems typically rely on static databases and generic recommendations that do not account for individual preferences, cultural dietary habits, or specific health conditions. Most existing systems focus primarily on calorie counting rather than comprehensive nutritional analysis and personalized meal planning.

Limitations of Existing Systems:

  • Limited personalization capabilities based on individual health profiles
  • Lack of AI-driven intelligent recommendations
  • Insufficient support for specific dietary preferences and restrictions
  • Limited integration with modern AI technologies
  • Basic user interface and user experience design
  • Inadequate administrative tools for system management
  • Limited support for cultural and regional food preferences

Research in the field of digital nutrition has highlighted the importance of personalized dietary interventions and the potential of artificial intelligence in improving nutritional outcomes [1]. Studies have shown that personalized nutrition recommendations can significantly improve user adherence to dietary guidelines compared to generic advice [2].

2.2 Proposed System

The Smart Diet Planner addresses the limitations of existing systems by incorporating advanced artificial intelligence capabilities and a comprehensive approach to nutrition management.   • AI-powered personalized meal planning using Google's Generative AI

The system architecture is designed to be modular and extensible, allowing for easy integration of additional AI services and features. The use of modern web technologies ensures optimal performance and scalability.

Chapter 4
System Design
4.1 System Perspective

The Smart Diet Planner follows a three-tier architecture pattern consisting of the presentation layer, business logic layer, and data access layer. This architectural approach ensures separation of concerns, maintainability, and scalability.

Presentation Layer: The presentation layer consists of web-based user interfaces built using HTML5, CSS3, and JavaScript. This layer handles user interactions, input validation, and data presentation. The responsive design ensures optimal user experience across different devices and screen sizes.

Business Logic Layer: The business logic layer is implemented using the Flask web framework and contains the core application logic, including user authentication, diet plan generation, nutritional calculations, and AI integration. This layer processes user requests, applies business rules, and coordinates with external services.

Data Access Layer: The data access layer utilizes SQLAlchemy ORM to interact with the database. This layer handles data persistence, retrieval, and ensures data integrity through proper validation and constraints.

The system integrates with Google's Generative AI service to provide intelligent diet planning capabilities. This external integration enables the system to generate personalized recommendations based on user profiles and preferences.

Detailed Design
5.1 Class Diagram

The class diagram represents the object-oriented design of the Smart Diet Planner system, illustrating the relationships between different entities and their attributes and methods.

Core Classes:

User Class: The User class represents system users and contains attributes such as user_id, username, email, password_hash, personal information (age, gender, height, weight), health preferences (activity_level, food_preferences, is_diabetic), and nutritional goals (daily_calorie_goal, protein_goal, carb_goal, fat_goal).

Food Class: The Food class represents food items in the system database with attributes including food_id, name, category, nutritional information (calories_per_100g, protein_per_100g, carbs_per_100g, fat_per_100g), and metadata (is_vegetarian, is_diabetic_friendly).

FoodLog Class: The FoodLog class tracks user food consumption with attributes such as log_id, user_id, food_id, quantity, consumption_date, and meal_type (breakfast, lunch, dinner, snack).

UserPreferences Class: This class stores detailed user preferences including dietary restrictions, disliked foods, and custom nutritional goals.

User Manual

Getting Started

System Access: The Smart Diet Planner can be accessed through any modern web browser by navigating to the application URL. Ensure you have a stable internet connection for optimal performance.

Account Registration:

  1. Click on the "Register" link on the homepage
  2. Fill in all required personal information fields
  3. Provide accurate health information including height, weight, age, and activity level
  4. Select your dietary preferences and any health conditions
  5. Set your nutritional goals (daily calories, protein, carbs, fats)
  6. Click "Create Account" to complete registration

Using the System

Logging In: Enter your username/email and password on the login page. Check "Remember Me" if using a personal device.

Dashboard Overview: The main dashboard provides a summary of your daily nutritional intake, progress toward goals, and quick access to key features.

Generating Diet Plans:

  1. Navigate to "AI Diet Plan" from the main menu
  2. Click "Generate New Plan" to create a personalized meal plan
  3. Review the AI-generated recommendations
  4. Save the plan or request modifications if needed

Logging Food Intake:

  1. Go to "Log Meal" from the dashboard
  2. Search for and select the food item
  3. Enter the quantity consumed
  4. Select the meal type (breakfast, lunch, dinner, snack)
  5. Click "Log Food" to save the entry

Viewing Progress: The "Progress" section provides detailed analytics on your nutritional intake, goal achievement, and historical trends.

Administrative Features

Admin Access: Users with administrative privileges can access additional features through the admin panel, including user management, food database management, and system analytics.

Troubleshooting

Common Issues:

  • Login Problems: Verify your credentials and ensure cookies are enabled
